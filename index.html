מדריך התקנה מתקדם

1. הורדת GIT

לחץ כאן להורדת GIT והתקן אותו.

2. הורדת Node.js (v20.15.1)

לחץ כאן להורדת Node.js והתקן אותו.

3. הורדת pnpm (v9.4.0)

הרץ את הפקודה הבאה ב-CMD או ב-PowerShell:

npm install -g pnpm@9.4.0

4. שיבוט המאגר

git clone https://github.com/coleam00/bolt.new-any-llm.git

5. התקנת תלויות

cd bolt.new-any-llm pnpm install

6. הגדרת קובץ הסביבה

שנה את שם הקובץ .env.example ל-.env.local ופתח אותו לעריכה.

7. הורדת Chrome Canary

לחץ כאן להורדת Chrome Canary והתקן אותו.

8. הפעלת הפרויקט

npm run dev

9. סיום

כל הכבוד! השלמת את תהליך ההתקנה.
